Cізге үй жұмыстарына арналған арнайы папка ашуды ұсынамын. Сосын әртүрлі үй жұмысын әртүрлы папкаға сақтаңыз. Сонда сізге жұмыс жасау ыңғайлы болады. 
Мысалы Үй жұмыстары -> Үй жұмысы 1 -> react app
			Үй жұмысы 2 -> react app
			тағы басқа
	
1-тапсырма: React Орнату және "Сәлем алем" деп шығару

node.js орнатқаныңызға қол жеткізіңіз ол үшін npm деп жазып көріңіз

npx create-react-app my-app пәрменін пайдаланып, Жаңа React қолданбасын жасаңыз. my-app қолданбасын қалаған атыңызбен ауыстырыңыз.

Қолданбаны жасағаннан кейін жасалған файлдар мен қалталардың құрылымын мұқият қарап шығыңыз.

Содан соң App.js - тың return  ішіндегісін жойып "Сәлем әлем" деп жазыңыз, сосын нәтижені тексеріңіз!
