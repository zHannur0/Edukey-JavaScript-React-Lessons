Тапсырма: React көмегімен калькулятор құру

Мақсаты: React көмегімен қарапайым калькулятор жасаңыз. Калькулятор сандарды енгізу үшін екі өрісті, "қосу" батырмасын және қосу нәтижесін көрсетуді қамтуы керек.

Нұсқаулар:

Create React App көмегімен жаңа React жобасын жасаңыз.

Калькулятор компонентін жасаңыз, ол калькулятор болады.

Калькулятор компонентінде:

Сандарды енгізу үшін екі input жасаңыз.
"Қосу" батырмасын жасаңыз.
Қосу нәтижесінің дисплейін жасаңыз.
Қосу нәтижесінің мәнін сақтау үшін userstate ілгегі арқылы күйді пайдаланыңыз.

Сандарды енгізу өрістеріне кіру үшін useRef ілмектерін пайдаланыңыз.
  const firstNumberRef = useRef(null);
  const secondNumberRef = useRef(null);
  const [sum, setSum] = useState(0);

"Қосу" батырмасын басқан кезде енгізу өрістерінен мәндерді алыңыз, оларды қосыңыз және нәтижемен күйді жаңартыңыз.

Қосу нәтижесін экранда көрсетіңіз.
