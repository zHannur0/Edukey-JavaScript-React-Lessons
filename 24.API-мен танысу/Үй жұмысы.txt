Үй тапсырмасы: API және HTTP сұрау теориясы

Астыда жазылған сұрақтарға өз беттеріңше жауап іздеп көріңдер!

API (Application Programming Interface) деген не және оның негізгі мақсаттары мен міндеттері қандай екенін сипаттаңыз.

HTTP (Hypertext Transfer Protocol) дегеніміз не және оның жұмыс принципі қандай?

Сіз HTTP сұрауларының қандай түрлерін білесіз (GET, POST, PUT, DELETE және т. б.) және олардың әрқайсысы не үшін қолданылады?

Сіз қандай http күй кодтарын білесіз? Олардың бірнешеуін атаңыз және олардың мағыналарын түсіндіріңіз.

HTTP сұраулары мен жауаптарында қандай деректер түрлері берілуі мүмкін? Мысалдар келтіріңіз.

REST (Representational State Transfer) дегеніміз не және ол қандай принциптерді ұсынады?

JSON (JavaScript Object Notation) дегеніміз не және оның API деректер алмасу контекстінде қандай ерекшеліктері бар?

Веб-қосымшаларды әзірлеуде API-мен жұмыс істеу үшін қандай құралдар мен кітапханалар қолданылады?

Неліктен API-мен жұмыс істеу кезінде қауіпсіздік принциптерін сақтау маңызды және әдетте қандай қауіпсіздік шаралары қолданылады?
