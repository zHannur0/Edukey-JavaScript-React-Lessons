Практикалық үй тапсырмасы: API-мен жұмыс

Мақсаты: сыртқы API-ге HTTP сұрауларын жасауды және қолданбада алынған деректерді өңдеуді үйрену.

Нұсқаулар:

API таңдаңыз: жұмыс істеу үшін жалпыға ортақ API таңдаңыз. https://github.com/public-apis/public-apis мына жердегі апилардың кез келгенін қолдана беріңіз.

React жобасын жасаңыз: Create React қолданбасын пайдаланыңыз немесе жобаңызды React бағдарламасына жинаңыз.

Компонент жасаңыз: таңдалған API-мен жұмыс істейтін жобаңызда жаңа компонент жасаңыз. Мысалы, оны ApiData деп атаңыз.

HTTP сұрауын жасаңыз: таңдалған API-ге GET сұрауын орындау үшін axios кітапханасын немесе стандартты fetch пайдаланыңыз. Сұрау пайдаланушылар, ауа райы, мақалалар және т. б. сияқты деректерді алуға байланысты болуы мүмкін.

Жауапты өңдеңіз: серверден алынған жауапты өңдеңіз, мысалы, экранда алынған деректерді шығару немесе басқа деректер әрекетін орындау. Деректерді көрсету үшін usestate ілгегі арқылы күйді пайдалануға болады.

Бұл практикалық тапсырманың дұрыс жауабы жоқ. Кез-келге апиды өздеріңізге ыңғайлылығына қарап жасай беріңіздер
