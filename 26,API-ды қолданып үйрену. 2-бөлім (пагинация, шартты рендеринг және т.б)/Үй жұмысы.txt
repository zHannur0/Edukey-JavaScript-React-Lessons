Сосын осы сұрақтарға жауап іздеп қараңыздар!

API және HTTP:

a. API дегеніміз не және ол веб-әзірлеуде не үшін қолданылады?

b. HTTP сұрауларының қандай түрлерін білесіз және олардың әрқайсысы не үшін қолданылады?

c. JavaScript көмегімен сыртқы API-ге GET сұрауын қалай орындауға болады?

Пагинация:

a. пагинация дегеніміз не және ол не үшін қолданылады?

b. пагинацияда қандай элементтер жиі кездеседі және олар нені білдіреді?

c. үлкен деректер жиынтығы үшін қандай пагинация стратегияларын ұсына аласыз?

Шартты көрсету:

a. react-те шартты көрсету дегеніміз не?

b. JSX-те шартты көрсетуді орындау үшін қандай оператор қолданылады?
