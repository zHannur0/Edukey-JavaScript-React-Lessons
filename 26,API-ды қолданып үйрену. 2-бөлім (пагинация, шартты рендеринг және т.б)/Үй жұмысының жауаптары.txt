Жауаптар:

API және HTTP:

A. API (Application Programming Interface) - бұл бағдарламалардың бір-бірімен өзара әрекеттесуіне мүмкіндік беретін ережелер мен хаттамалар жиынтығы. Ол қолданбалар арасында деректерді бөлісу үшін қолданылады.

b.HTTP сұрауларының түрлеріне GET (деректер сұрауы), post (деректерді серверге жіберу), PUT (сервердегі Деректерді жаңарту) және DELETE (сервердегі деректерді жою) кіреді. Олар әртүрлі операциялар үшін қолданылады.

c. JavaScript-те API-ге GET сұрауын орындау үшін fetch() немесе Axios сияқты кітапханаларды пайдалануға болады.

Пагинация:

а.Пагинация - оңай көрсету үшін үлкен деректер жинағын шағын беттерге бөлу процесі.

b. пагинация элементтеріне Алдыңғы бет және келесі бет түймелері, бет нөмірлері, ағымдағы бет туралы ақпарат және беттердің жалпы саны кіреді.

c. пагинация стратегиялары жылжу кезінде беттерді жүктеуді (шексіз айналдыру), беттің төменгі жағындағы бет нөмірлерін немесе беттегі элементтер санын таңдауды қамтуы мүмкін.

Шартты көрсету:

a. Шартты көрсету-бұл белгілі бір шарттар негізінде нені көрсету керектігін шешу процесі.

b. JSX-те шартты рендерингті орындау үшін тернарлы оператор (Шартты оператор) қолданылады.
